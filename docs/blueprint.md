# **App Name**: SpeseJournal

## Core Features:

- Expense Recording: Record expenses with descriptions, amounts, and categories.
- Budget Setting: Set monthly budgets for different spending categories.
- Automatic Categorization Tool: Use AI to automatically categorize expenses based on transaction details; AI determines when to categorize each transaction, by consulting its pre-programmed list of when automatic categorization should happen.
- Report Generation: Generate visual reports of spending patterns over time.
- Subscription Tracking: List ongoing subscriptions with renewal dates and amounts.

## Style Guidelines:

- Primary color: Deep purple (#673AB7), evokes financial trustworthiness.
- Background color: Light gray (#F5F5F5), subtly integrates with the purple.
- Accent color: Teal (#009688) for calls to action and highlights; provides a vibrant contrast.
- Body and headline font: 'Inter', a sans-serif for clear readability and a modern look.
- Simple, line-based icons to represent expense categories.
- Clean, card-based layout to organize information efficiently.
- Subtle transitions and animations to enhance user interaction and provide feedback.
